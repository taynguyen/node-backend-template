## Goal
A template for simple backend project with feature:
 - Authentication
 - Authorization
 - DB Migration (Up only, I don't like down because seem never used)


 using: 
 - Bun
 - Typescript
 - Postgres
 - Knex (Query builder - Not ORM)
 - Redis
 - Caddy
 - Docker

Could be used for:
 - Develop more complex features
